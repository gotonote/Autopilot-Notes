# 9.4 蔚来/小鹏/理想智驾方案对比

## 9.4.1 概述

蔚来、小鹏、理想作为中国新势力造车三强，在智能驾驶领域各有特色。本章节从技术架构、硬件配置、功能体验等维度进行全方位对比分析。

## 9.4.3 蔚来 NAD 方案

### 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    蔚来 NAD 架构                         │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────┐   │
│  │              Adam 超算平台 (4×Orin X)            │   │
│  └─────────────────────────────────────────────────┘   │
│                         ↓                               │
│  ┌──────────────┬──────────────┬──────────────┐       │
│  │  感知模块     │  规划模块    │  控制模块     │       │
│  │ (Aquila)     │  (Supreme)  │  (Control)   │       │
│  └──────────────┴──────────────┴──────────────┘       │
└─────────────────────────────────────────────────────────┘
```

### 技术特点

1. **超算平台 Adam**
   - 4颗 NVIDIA Orin X 芯片
   - 1016 TOPS 算力冗余
   - 独立安全芯片

2. **感知系统 Aquila**
   - 1550nm激光雷达（最远500m）
   - 7颗800万像素摄像头
   - V2X车路协同（可选）

3. **功能体验**
   - 高速NOP+：完全覆盖高速场景
   - 城市NAD：逐步开放中
   - 智能泊车：APA + RPA

### 代表车型

- ET7、ES7、EC7
- ES8、ES6、EC6（换代车型）
- ET5、ET5T

2. **XPlanner 规划决策**
   - 神经网络+规则混合
   - 拟人化驾驶行为
   - 博弈决策能力

3. **功能体验**
   - **城市NGP**：全国范围逐步开放
   - **高速NGP**：全覆盖
   - **VPA停车场记忆泊车**：跨楼层记忆泊车

- G6、G9
- P7i、P5
- X9

## 9.4.8 总结

| 品牌 | 核心优势 | 适合人群 |
|:-----|:---------|:---------|
| 蔚来 | 硬件领先、服务体系完善 | 追求高端体验、换电便利 |
