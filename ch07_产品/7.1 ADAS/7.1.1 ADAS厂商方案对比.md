## 1️⃣ 华为 ADS

### 1.1 技术架构

华为 ADS (Autonomous Driving Solution) 采用 MDC 平台 + 激光雷达融合方案。

```
华为 ADS 架构
┌─────────────────────────────────────────────────────────┐
│                    硬件平台                              │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐     │
│  │  MDC 810    │  │  昇腾910    │  │  激光雷达   │     │
│  │ (400+ TOPS) │  │  (AI算力)   │  │  192线      │     │
│  └─────────────┘  └─────────────┘  └─────────────┘     │
│         ↓                ↓                ↓             │
│  ┌─────────────────────────────────────────────────────┐│
│  │                    软件算法                          ││
│  │  感知 → 融合 → 预测 → 规划 → 控制                     ││
│  └─────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────┘
```

### 1.2 核心能力

- **感知能力**: 激光雷达+视觉融合，支持动/静态目标检测
- **场景覆盖**: 城市NCA、高速NCA、泊车APA/AVP
- **安全冗余**: 转向、制动、电源多重冗余

### 1.3 合作车型

- 问界 M5/M7/M9
- 阿维塔 11/12
- 极狐 αS/αT

## 2️⃣ 小鹏 XNGP

### 2.1 技术架构

小鹏 XNGP (Xiaoming Guided Navigation Pilot) 采用双 Orin X + 激光雷达方案。

```
小鹏 XNGP 硬件配置
┌─────────────────────────────────────────────────────────┐
│  计算平台: 英伟达 Orin X × 2 (508 TOPS)                │
│  传感器:                                                 │
│    - 激光雷达: 速腾聚创 M1 × 2                          │
│    - 摄像头: 800万像素 × 12                             │
│    - 毫米波雷达: 博世五代 × 5                          │
│    - 超声波雷达: 12颗                                   │
└─────────────────────────────────────────────────────────┘
```

### 2.2 核心能力

- **城市NGP**: 覆盖一线城市核心路段
- **高速NGP**: 高速NOA完全体的代表
- **泊车能力**: VPA 记忆泊车、APA 自动泊车
- **无图化**: 逐步摆脱高精地图依赖

### 2.3 迭代历程

| 版本 | 发布时间 | 核心能力 |
|:-----|:---------|:---------|
| XNGP 3.0 | 2022.10 | 高速NGP |
| XNGP 4.0 | 2023.06 | 城市NGP |
| XNGP 5.0 | 2024.x | 无图NOA |

## 3️⃣ 蔚来 NAD

### 3.1 技术架构

蔚来 NAD (NIO Autonomous Driving) 采用四 Orin X 超高算力方案。

```
蔚来 NAD 硬件配置
┌─────────────────────────────────────────────────────────┐
│  计算平台: 英伟达 Orin X × 4 (1016 TOPS)               │
│  传感器:                                                 │
│    - 激光雷达: 图达通 Falcon × 1                        │
│    - 摄像头: 800万像素 × 11                             │
│    - 毫米波雷达: 5颗                                    │
│    - 超声波雷达: 12颗                                   │
│    - V2X: 5G V2X                                       │
└─────────────────────────────────────────────────────────┘
```

### 3.2 核心能力

- **算力优势**: 1016 TOPS 行业领先
- **感知冗余**: 多种传感器融合
- **换电体系**: 补能体系优势
- **BaaS**: 电池租用降低门槛

### 3.3 争议与挑战

- 硬件预埋导致成本较高
- 软件订阅模式用户接受度

理想 AD Max 硬件配置

│  传感器:                                                 │

│    - 毫米波雷达: 1颗前向 + 4颗角雷达                   │
│    - 超声波雷达: 12颗                                   │

### 4.2 核心能力
