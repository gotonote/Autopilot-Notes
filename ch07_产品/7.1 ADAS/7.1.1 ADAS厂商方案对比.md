## 1️⃣ 华为 ADS

### 1.1 技术架构

华为 ADS (Autonomous Driving Solution) 采用 MDC 平台 + 激光雷达融合方案。2026年已升级至 **ADS 4.0**。

> **最新动态**: 华为 ADS 4.0 将于2026年推出，疑似采用 VLA (Vision-Language-Action) 大模型架构

```
华为 ADS 架构
┌─────────────────────────────────────────────────────────┐
│                    硬件平台                              │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐     │
│  │  MDC 810    │  │  昇腾910    │  │  激光雷达   │     │
│  │ (400+ TOPS) │  │  (AI算力)   │  │  192线      │     │
│  └─────────────┘  └─────────────┘  └─────────────┘     │
│         ↓                ↓                ↓             │
│  ┌─────────────────────────────────────────────────────┐│
│  │                    软件算法                          ││
│  │  感知 → 融合 → 预测 → 规划 → 控制                     ││
│  └─────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────┘
```

### 1.2 核心能力

- **感知能力**: 激光雷达+视觉融合，支持动/静态目标检测
- **场景覆盖**: 城市NCA、高速NCA、泊车APA/AVP
- **安全冗余**: 转向、制动、电源多重冗余
- **ADS 3.0**: 2026年最新版本，端到端大模型架构
- **ADS 4.0 (即将)**: 疑似 VLA 大模型架构，预计2026年发布

### 1.3 合作车型

- 问界 M5/M7/M9
- 阿维塔 11/12
- 极狐 αS/αT
- 享界 S9

## 6️⃣ 特斯拉 FSD 最新进展 (2026)

### 6.1 FSD v14.x

- **最新版本**: FSD v14.x 将于2026年在Ride AI 2026大会上提供试乘体验
- **累计里程**: 特斯拉FSD累计行驶里程已突破**80亿英里**
- **技术路线**: 纯视觉方案 + 端到端神经网络
- **AI4硬件**: HW4.0 + 16GB RAM
- **V14 核心特点**: 
  - Scaling Law 在车端"暴力美学"的初次展现
  - 从规则驱动到数据驱动的重大进步
  - 计划2026年部署至Robotaxi fleet，取消随车安全员

- **量产时间**: 2026年4月正式投产

## 8️⃣ L3 级自动驾驶 (2026)

### 政策动态

- **L3政策即将落地**: 2026年L3级有条件自动驾驶法规即将正式落地
- **应用场景**: 在法规允许的路段，驾驶员可合法脱手脱眼
- **责任划分**: 明确车企责任与驾驶员责任边界

### 技术收敛

- **方案收敛**: L3级技术方案将逐渐收敛并实现规模化落地
- **场景覆盖**: 高速场景 → 城市道路 → 特定区域
- **玩家**: 华为、小鹏、蔚来、理想等头部厂商积极布局
