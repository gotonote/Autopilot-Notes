# 仿真

自动驾驶仿真平台是算法开发和测试的重要工具，本章节介绍主流仿真平台的使用方法。

## 目录

### 8.2.1 [CARLA 仿真](./8.2.1%20Carla仿真/readme.md)

开源自动驾驶仿真平台，支持：
- 多元城市场景构建
- 真实物理引擎
- 多种传感器模拟
- Python/C++ API

### 8.2.2 [PreScan 仿真](./8.2.2%20PreScan仿真/readme.md)

西门子自动驾驶仿真平台，特点：
- MATLAB/Simulink 集成
- 强大的传感器仿真
- 快速原型开发
- 支持 OpenSCENARIO/OpenDRIVE

## 仿真平台对比

| 平台 | 特点 | 适用场景 |
|------|------|----------|
| CARLA | 开源、可定制性强 | 算法研究、原型开发 |
| PreScan | 工业级、Simulink集成 | OEM 量产开发 |
| 51SimOne | 国产、云端仿真 | 快速验证 |
| 腾讯TAD Sim | 腾讯生态、游戏级渲染 | 场景仿真 |

## 学习建议

1. **入门推荐 CARLA** - 开源免费，文档丰富
2. **工业应用选 PreScan** - 与 MATLAB 无缝集成
3. **实战结合理论** - 边学边做效果最好
